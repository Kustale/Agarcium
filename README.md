# Agarcium
MapleStory 2 Proxy DLL & Patcher

Usage/Install:
- Rename NxCharacter64.dll to NxCharacter64.dll.bak (required)
- Move NxCharacter64.dll (Proxy) & AgarciumClient.dll to folder
- Launch the game exe with params:
- 
        --nxapp=nxl
  
		--username=<username>
  
		--password=<password>
  
		--title=<title>
  
		--ip=<ip>
  
		--port=<port>
  
		--mods=<path>
  
		--autologin
  
		--console
